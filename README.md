# NewProject
基于EGL语言的小学数学智能教学系统

# 项目介绍
>项目来源于辽宁省国家自然基金项目
>项目产生两篇相关论文，一个实体项目。
>项目基于第四代模板编程语言，加入了深度学习算法为系统资源的准确推荐提供保障。
